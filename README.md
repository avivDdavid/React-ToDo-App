# אפליקציית ניהול משימות (Task Manager)

פרויקט מסכם בקורס React.
האפליקציה משמשת לניהול משימות (Todo List) וכוללת יכולות עריכה, סינון ושמירת נתונים, עם ממשק משתמש מותאם אישית.

## תכונות מרכזיות
* **ניהול משימות:** יצירה, עריכת תוכן ומחיקת משימות.
* **ניהול סטטוס:** שינוי סטטוס משימה (הושלמה/פעילה) בלחיצה.
* **סינון תצוגה:** אפשרות לסינון הרשימה לפי סטטוס: הכל, פעילות, או הושלמו.
* **שמירת נתונים (Persistence):** סנכרון אוטומטי מול LocalStorage לשמירת המידע בין רענוני דף.
* **ממשק משתמש:** עיצוב מותאם אישית (Custom CSS) ללא שימוש בספריות עיצוב חיצוניות.

## הוראות התקנה והרצה

1. שכפול הפרויקט:
   git clone [repository-url]

2. התקנת תלויות (Dependencies):
   npm install

3. הרצת שרת פיתוח מקומי:
   npm run dev

## ארכיטקטורה ומבנה קומפוננטות

בהתאם לדרישות הפרויקט, האפליקציה מחולקת לרכיבים הבאים לצורך הפרדת אחריות (Separation of Concerns):

* **App:** הקומפוננטה הראשית. אחראית על ניהול ה-State המרכזי (משימות, פילטרים), לוגיקת השמירה (LocalStorage) ופונקציות הליבה (הוספה, מחיקה, עריכה).
* **TodoForm:** רכיב קלט. אחראי על טופס הוספת המשימות ואימות הקלט הראשוני.
* **TodoFilter:** רכיב בקרה. מציג את כפתורי הסינון ואת מונה המשימות הפעילות.
* **TodoList:** רכיב תצוגה. מקבל את רשימת המשימות המסוננת ומרנדר את הפריטים.
* **TodoItem:** רכיב בודד. מציג את פרטי המשימה ומאפשר אינטראקציה (סימון, כניסה למצב עריכה, מחיקה).
